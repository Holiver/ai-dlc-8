## 🎯 项目目标

构建一个完整的员工福利电商系统，支持：
- 员工使用积分兑换福利产品
- 管理员管理员工、产品、积分和订单
- 多语言支持（中英双语）
- 响应式设计（支持移动端和桌面端）

## 🏗️ 架构设计

### 后端架构（三层架构）
```
┌─────────────────────────────────────┐
│         API Layer (Handler)         │
│  - 请求处理                          │
│  - 输入验证                          │
│  - 响应格式化                        │
└─────────────────────────────────────┘
                 ↓
┌─────────────────────────────────────┐
│       Business Layer (Service)      │
│  - 业务逻辑                          │
│  - 事务处理                          │
│  - 数据转换                          │
└─────────────────────────────────────┘
                 ↓
┌─────────────────────────────────────┐
│      Data Layer (Repository)        │
│  - 数据访问                          │
│  - CRUD 操作                         │
│  - 查询构建                          │
└─────────────────────────────────────┘
                 ↓
┌─────────────────────────────────────┐
│          Database (MySQL)           │
└─────────────────────────────────────┘
```

### 前端架构（组件化架构）
```
┌─────────────────────────────────────┐
│          Pages (页面组件)            │
│  - 业务页面                          │
│  - 路由组件                          │
└─────────────────────────────────────┘
                 ↓
┌─────────────────────────────────────┐
│       Components (通用组件)          │
│  - UI 组件                           │
│  - 可复用组件                        │
└─────────────────────────────────────┘
                 ↓
┌─────────────────────────────────────┐
│        Services (服务层)             │
│  - API 调用                          │
│  - 数据处理                          │
└─────────────────────────────────────┘
                 ↓
┌─────────────────────────────────────┐
│         Backend API (后端)           │
└─────────────────────────────────────┘
```
